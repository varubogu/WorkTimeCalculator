# WorkTime Calculator

SES エンジニア向けの作業時間管理ツールです。月間・年間の勤務時間をブラウザ内で記録し、設定した目標範囲との差分を確認できます。

## 特徴

- 日別の開始時刻、終了時刻、休憩時間、休暇を記録
- 月間・年間の合計時間と目標範囲を表示
- 月間・年間の残業時間を目標範囲付きで表示
- 作業時間の表示形式を `7:30` と `7.50h` で切り替え
- 平日への定時一括入力
- 土日・祝日を含む休日への勤務入力
- 日本の祝日を Holidays JP API から取得し、取得できない場合は同梱データへフォールバック
- 日本語 / 英語、ライト / ダークテーマ対応
- 作業時間ファイルの CSV / JSON / YAML 書き出しと読み込み
- 設定ファイルの JSON / YAML 書き出しと読み込み
- 設定の有効期間を定義し、期間ごとに定時時間・開始時刻・休憩・入力刻みを切り替え

勤務データと設定は `localStorage` に保存されます。サーバーへ勤務データは送信されません。

## 開発

```bash
bun install
bun run dev
```

開発サーバーは Vite が起動します。既定のポートは 3000 です。

## ビルド

```bash
bun run build
```

`dist/` に本番用ファイルを生成します。Bun の静的サーバーで配信する場合は次を実行します。

```bash
bun start
```

## テスト

```bash
bun run test
bun run test:e2e
```

Playwright のブラウザが未インストールの場合は、初回のみ `bunx playwright install` を実行してください。詳しくは [docs/testing.md](./docs/testing.md) を参照してください。
